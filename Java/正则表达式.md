# 正则表达式

## 基本概念
### 分组捕获
(...)
### 分组不捕获
(?:...)
### 肯定顺序环视
(?=aaa) 从左往右扫描
### 肯定逆序环视
(?<=aaa)    从右往左扫描
### 否定顺序环视
(?!aaa)
### 否定逆序环视
(?< !aaa) (< 与 !中间的空格是不能存在的，不过不加md编辑时显示有点问题)

### 解析正则表达式：
```
"^(\\d{4}-\\d{2}-\\d{2} \\d{2}:\\d{2}:\\d{2})\\s*" +
        "\\[(\\w+)]\\s*" +
        "\\[([-\\d\\w]+)]\\s*" +
        "([\\w\\d\\[\\]]+(?= - Req))" +
        "[- ]+Req: (\\w+)\\s*" +
        "([-\\w\\d:./]+)\\s*" +
        "([-\\w\\d=., &}{_\\u4e00-\\u9fa5]+)\\s*" +
        "Resp: Result:\\s*(.+(?=\tCosts))\\s*" +
        "Costs:([\\w\\d]+)$"
```
